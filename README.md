A simple CLI tool to quickly see package stats from Packagist.

## Installation

```
composer global require jaybizzle/statagist
```

## Usage
```
statagist vendorname/packagename
```

**Output**
```
### vendorname/packagename ###
Downloads
Today............. 3,841
Last 30 days...... 94,621
Total............. 383,033
```
